## Investigation Summary

The investigation revealed that the attacker performed the following activities:

- Network reconnaissance
- SQL injection attempts
- Repeated login attempts

These activities were successfully detected by the IDS and confirmed through SIEM log analysis.

## Key Evidence

Evidence collected during the investigation:

- IDS alerts generated by Snort
- Apache access logs showing malicious requests
- Splunk queries identifying suspicious patterns

These artifacts were used to reconstruct the attack timeline and validate detection mechanisms.
